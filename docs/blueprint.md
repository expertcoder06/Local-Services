# **App Name**: Local Services

## Core Features:

- Phone Verification: Verify user phone number using OTP for security and identity validation.
- Location-Based Service Discovery: Detect user location and display nearby service providers within a 5-10 km radius.
- Service Request Posting: Allow customers to post service requests with details such as budget, urgency, and location. Utilize a generative AI tool that evaluates budget, urgency, and other service request details to generate suitable bids that attract quality local service providers.
- Real-Time Bidding System: Enable service providers to bid on service requests in real time.
- Provider Profiles: Create profiles for service providers to showcase their portfolio, gallery, ratings, and reputation score.
- Escrow-Style Payment Flow: Implement an escrow-style payment system to hold payments until the service is completed, then release funds to the provider or offer a refund if necessary.
- Reputation Scoring: Calculate reputation scores for service providers based on ratings, job completion rate, and response time.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to inspire trust and reliability.
- Background color: Light gray (#F0F4F7), a desaturated, brighter shade of the primary.
- Accent color: Muted violet (#80759B), providing contrast and signaling interactive elements.
- Body and headline font: 'PT Sans' for a balance of modern look and approachability.
- Use clean, minimalist icons to represent service categories and actions.
- Design a mobile-first, responsive layout that adapts to different screen sizes.
- Incorporate subtle animations to enhance user experience and provide feedback on interactions.
